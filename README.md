# Site modelo profissional — Casa Bruma

Este é um site fictício, criado para ser mostrado por WhatsApp como exemplo de site para comércios locais.

## O que mudou nesta versão

- Visual mais profissional e menos “cara de IA”
- Fotos reais externas via Unsplash
- Tipografia mais refinada
- Layout mais editorial
- Textos mais naturais
- Sem excesso de emoji, gradiente e elementos genéricos
- Site com cara de cliente real

## Como mostrar ao cliente sem ele baixar

Você precisa hospedar o HTML para gerar um link público.

Opções fáceis:
1. Netlify Drop
   - Entre em app.netlify.com/drop
   - Arraste a pasta ou o ZIP do site
   - Ele gera um link público na hora

2. Vercel
   - Suba o projeto
   - Ele gera um link público

3. GitHub Pages
   - Crie um repositório
   - Suba o index.html
   - Ative Pages nas configurações

## Trocar WhatsApp

No arquivo index.html, procure por:
5512999999999

Troque pelo número real do cliente, no formato:
55 + DDD + número

Exemplo:
5512987654321
